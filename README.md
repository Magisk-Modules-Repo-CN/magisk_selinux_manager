# Magisk SELinux 管理器
一个专门用于SELinux的操作和分析Magisk模块

## 特征
  * 启动时设置SELinux 强制或允许(Enforcing or Permissive)

## 依赖
  * Magisk v15.3+
  * Android v4.3+ 设备
  * 可调整SELinux的内核
